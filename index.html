Adding things

Put it on the board in seconds

Hit the Add button on any screen. Pick Service or Activity, choose the level, give it a name and a date. Done.

Add
  • It opens already set to wherever you're standing — add from the Service tab and it's a service item
  • Add a point person so everyone knows who owns it
  • Note what's needed — trucks, tools, a grill, strong backs
Keeping it moving

Planned → Promoted → Done

One tap moves an item forward. Promote it when you've announced it. Mark it done when it's over.

Planned → Promoted → Done

When you mark something done, it asks how many brethren showed up — and for service, the hours. That's how the board remembers what the quorum actually accomplished.

The weekly rhythm

How a presidency runs it

  • In presidency meeting, open The Board — what's open, what's next
  • Promote this week's items and name a point person for each
  • After an event, mark it done and log who came
  • Sunday: copy the announcements into the quorum text
  • Every quarter, check the Timeline for gaps and plan ahead

Keep it a planning board — what you planned, did, and who you reached. It's not a ministering record. Pastoral notes stay out.
